# Ignite Redux Persist Plugin

This plugin adds support for redux-persist to an [Ignite Andross](https://github.com/infinitered/ignite-andross) project.

## Install

```
ignite add redux-persist
```

## Uninstall

```
ignite remove redux-persist
```

## Contributing
